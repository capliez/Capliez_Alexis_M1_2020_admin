### Objectives

### Prerequisites

What you need to use it:

### Installation

Step to install:
  1. ``` git clone https://github.com/capliez/Capliez_Alexis_M1_2020_admin ```
  2. ``` chmod a+x FILE_NAME.sh ```

### How to use the Script

In the terminal execute :
./FILE_NAME.sh [arguments]
